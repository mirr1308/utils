1. 개요

이 웹에디터는 스케줄표를 쉽게 작성하기 위해 만들어진 자동 확장/커스텀 보조 도구입니다.
핵심기능
- 기존 Table 삽입 및 분석
- 커스텀 단축 버튼 설정
- 달력 자동 변환
- 쉽게하는 줄 확장
- 쉽게하는 칸 분할

전체 화면. 커서가 오른쪽 창에 있지 않아 상단 툴바의 불러오기/내보내기/코드복사/도움말을 제외하곤 비활성화 된 상태다.


왼쪽은 HTML 코드가 나타나는 텍스트 에디터이고, 오른쪽은 미리보기가 가능합니다. 상단의 툴바는 일부 기능을 제외하고 우측 작업 공간에서만 사용하실 수 있습니다.

사용자 편의를 위한 부가 기능
- 왼쪽 텍스트 에디터에서 Ctrl+Z를 누르면 되돌리기가 가능
- 오른쪽 프리뷰 에디터에서 Ctrl+마우스 휠을 돌리면 부분 확대가 가능(오른쪽 한정).
- 현재 작업 위치를 알려주는 마커
- 각종 글자 꾸미기 기능 탑재

2. 데이터 분석/적용

분석할 Table 양식을 붙여넣어 주세요. 자동 정제 기능이 있어 1일부터 31일까지 작성한 스케줄표를 넣어도 샘플 코드를 추출합니다.
저장을 누르면 캐시와 로컬 스토리지에 저장되어 컴퓨터를 껐다 켜도 데이터가 남아 있게 됩니다. 해당 텍스트창의 소스 코드를 지우고 저장을 누르면 완전히 삭제됩니다.

디자인을 결정 못 했으면 AI한테 각종 요구사항을 말해봅시다

위의 헤드는 그대로, td 설정도 그대로 유지하고, 배경색이 하나일 경우 1일에 하나, 배경색이 두 개일 경우 1일·2일 두 개로 적용합니다. 날짜에 두 줄이 남아 있을 때, 날짜별 구분 테두리와 날짜 내부 구분 테두리가 그대로 유지되면 완성!

샘플코드는 칸 안에 들어갈 내용을 제외하고 모든 디자인을 그대로 남깁니다. <table></table> 밖의 요소들은 지워지며(div 등등), 안의 요소도 지워집니다.
스케줄표는 반드시 1일부터 시작해야합니다. '1 수' 'Mon 1' 다 되나, 1 제외 다른 숫자는 허용하지 않습니다.

3. 줄 확장

분석된 샘플코드를 자동으로 불러와 1일부터 31일까지 만들어주는 기능입니다. 매월 바뀌는 스케줄표 디자인 바꾸기, 쉽게 버튼 하나로 해결하세요.

1일부터 30일, id 생성, 색상까지 다 넣은 모습


'기간 설정'에서 시작일과 종료일만 입력하고 저장을 눌러도 생성됩니다.

월 말일 기준으로 28일, 29일, 30일, 31일이 있습니다. 줄 확장기능은 table style을 그대로 상속, 샘플코드를 기반으로해 1일 ->2일, 2일 ->3일 등 일자가 넘어갈 때의 테두리를 상속하기때문에 혹시나 월 말일의 tr과 td에서 border-bottom 테두리만 따로 설정해놨다면 별도로 말일의 style을 수정하셔야 합니다.


날짜 ID 자동 생성은 캘린더 연동을 처음 하시는 분들께 권장합니다. day를 입력하면 자동으로 day1, day2, …, day31까지 생성되며, 입력하지 않아도 1, 2, …, 31 형식으로 생성됩니다.
이미 td id가 있는 경우, 샘플 코드를 분석하여 자동으로 id가 할당됩니다.

날짜 칸 색상에서는 #6자리, #3자리, red, gray, blue 등을 입력하면 자동으로 적용됩니다. 여러 색을 입력할 경우 반드시 쉼표로 구분해 주세요.
gray 입력 → 1, 2, … 해당 숫자열 전체가 gray 색상으로 변경됨
#333, red → 선택한 달 기준, 일요일에 해당하는 날짜만 red이고 나머지는 #333 색상으로 변경됨
#333, blue, red → 선택한 달 기준, 토요일은 파란색, 일요일은 빨간색, 나머지는 #333으로 변경됨

그 외 부가 기능으로 샘플 코드 날짜 칸에 '월', '월요일', 'Mon', 'MON', 'Monday'를 함께 입력해 두면 자동으로 생성됩니다. 단, 샘플 코드 기준으로 1일에 색상이 있으면 해당 색상으로 전체 줄 확장이 적용되므로, 첫째 날에 색이 있을 경우 날짜 칸 색상 변경 기능을 이용해 주세요.

자동 색상 변경을 사용할 때 샘플 코드에 요일과 숫자가 함께 있을 경우, 해당 월 기준으로 일치해야 합니다. 2026년 4월 기준으로 샘플 코드는 '1', '수', 'Wed'처럼 작성되어야 합니다. '일'이나 'Thu' 등 1일의 요일과 일치하지 않으면 자동 색상 변경을 사용할 수 없습니다.


중간 날짜 입력해 필요한 부분만 생성, 편집하세요.


샘플 코드는 한 번 저장해 놓으면 컴퓨터를 껐다 켜도 분석 툴을 누르지 않고 사용할 수 있습니다.

텍스트 에디터에 아무것도 없고, 줄 확장을 시작일·종료일 없이 누를 경우 → 샘플 코드로 1일~31일까지 자동 생성됩니다.
텍스트 에디터에 10일까지 작성한 스케줄표를 붙여넣고(10일이 최하단), 줄 확장을 누를 경우 → 기존 스케줄표 뒤에 이어서 추가됩니다.
11일~13일 등 특정 구간의 스케줄표만 작성하고 싶은 경우, 시작일 11일, 종료일 13일을 입력한 뒤 줄 확장을 누르면 → <tr> 행을 제외한 나머지는 모두 회색 처리되고, tr행만 코드 복사가 가능합니다. 스케줄표가 길어져 버벅거리는 것을 방지하기 위한 부분 편집 기능입니다. 쉽게 복사해서 붙여넣으세요.

기본형 생성은 달력을 만들어드립니다. 매우 기본적인 형태의 달력이며, 스케줄표에 달력을 넣는 이유는 내부 링크를 사용하기 위함이므로 td id로 연결되는 자동 링크도 함께 생성됩니다. #user_content_1 형식으로 생성되기 때문에, 스케줄표에서 줄 확장을 사용할 경우 샘플 코드에 td id="user_content_1"을 추가해 주세요.

26년 12월 달력이 됩니다

7. 링크 삽입/변경

일반적인 링크 기능과 동일합니다. td의 id 칸 수정 기능은 필요할 경우 사용하시면 됩니다. 밑줄 제거는 하이퍼링크에 표시되는 밑줄 스타일이 보이지 않도록 처리한 것입니다. 필요하시면 해제해서 사용하세요.

8. 불러오기/내보내기/코드복사

불러오기 - JSON 파일로 저장된 샘플 코드와 커스텀 툴바 설정을 불러와 덮어씁니다.
내보내기 - 현재 임시 저장된 샘플 코드와 커스텀 툴바 설정을 JSON 파일로 내보냅니다.
코드 복사 - HTML 코드를 복사합니다. 줄 확장 기능을 이용해 특정 구간만 편집할 경우, tr 행만 따로 복사할 수 있습니다.

샘플 코드와 커스텀 툴바 설정은 캐시와 로컬 스토리지 두 곳에 저장되지만, 만일의 상황에 대비해 백업을 권장합니다.
